Професійний профіль

Досвідчений IT-спеціаліст з глибоким розумінням адміністрування серверних систем та мережевої інфраструктури. Останнім часом активно розвиваюся в напрямку кібербезпеки (Cybersecurity) та моніторингу безпеки (SOC). Маю практичний досвід роботи з віртуалізацією, криптографічним захистом та пошуком вразливостей.

Компанія "SD"

Період: Теперішній час

  • Моніторинг інцидентів безпеки та виконання обов'язків у межах SOC.
  • Підтримка та забезпечення кібербезпеки (КБ) підприємства.
  • Адміністрування та оновлення критичної інфраструктури (успішне оновлення vCenter Server з версії 7.0.2 до 7.0.3t).
  • Діагностика та усунення несправностей серверного обладнання (робота з помилками Kernel-Power, відновлення конфігурацій систем моніторингу EDAN MFM CMS).
  • Налаштування та керування середовищами віртуалізації VMware.

Технічні навички

  • Кібербезпека: Робота з Web Security Academy (PortSwigger), аналіз Reflected XSS, тестування вразливостей.
  • Криптографія: Робота з PGP (генерація ключів, обмін, шифрування повідомлень).
  • Системне адміністрування: VMware vCenter, Windows Server, Linux.
  • Програмування та скриптинг: Написання скриптів на Python для автоматизації завдань.
  • Мережі та архітектура: Глибоке розуміння мережевих протоколів та архітектури процесорів.

Освіта та самоосвіта

  • Вивчення стандартів та норм з охорони праці (Labor Protection).
  • Практичні лабораторні роботи з Web Security та Penetration Testing.
  • Курси та тренінги з адміністрування серверних систем та кібербезпеки.
